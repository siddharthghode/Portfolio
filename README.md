# Siddharth Ghode - Developer Portfolio

A modern, responsive portfolio website showcasing my work as a Python & Django developer. Built with clean HTML, CSS, and vanilla JavaScript featuring smooth animations and interactive elements.

## 🚀 Live Demo

[View Portfolio](https://siddharthghode.github.io/portfolio) 

## ✨ Features

- **Responsive Design** - Optimized for all devices and screen sizes
- **Smooth Animations** - Custom CSS animations and scroll-triggered reveals
- **Interactive Elements** - Custom cursor, hover effects, and skill progress bars
- **Single Page Application** - Fast loading with all content in one page
- **Contact Form** - Functional contact form with success feedback
- **Modern UI** - Dark theme with cyan accents and glassmorphism effects

## 🛠️ Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Custom properties, flexbox, grid, and animations
- **Vanilla JavaScript** - DOM manipulation and interactivity
- **Google Fonts** - Syne and JetBrains Mono typefaces

## 🏃‍♂️ Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/siddharthghode/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser:
   - Double-click the file
   - Or use a local server: `python -m http.server 8000`

3. Navigate to `http://localhost:8000` (if using server)
